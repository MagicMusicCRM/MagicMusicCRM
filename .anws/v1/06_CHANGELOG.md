# Changelog (v1)

Все локальные изменения в рамках архитектуры v1 фиксируются здесь. Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.1.0] - 2026-04-14

### Added
- **Infrastructure**: Полный переход на Custom Domain **`api.magic-music.org`** (Cloudflare Proxy) для повышения стабильности API.
- **Android**: Включен механизм **Core Library Desugaring** для поддержки современных API уведомлений на старых версиях Android.
- **Messenger**: Система «Заглушить» (Mute) чат с персистентным хранением в Supabase и фильтрацией пуш-уведомлений на стороне сервера.
- **Messenger**: Вкладки «Медиа», «Файлы», «Ссылки» в профиле чата теперь полностью функциональны и загружают данные в реальном времени.
- **UI/UX**: Поддержка открытия профиля через клик по аватарке в заголовке чата.

### Fixed
- **Database**: Глобальное исправление ошибок типов `user_role` в тригграх через явное приведение к `::text` и использование `SECURITY DEFINER`. Решена проблема переполнения стека (54001).
- **Notifications**: Исправлена несовместимость с `flutter_local_notifications` 18.x (позиционные аргументы в API).
- **Messenger**: Финализированы исправления Race Conditions при переключении чатов.
- **Messenger**: Исправлена ошибка синхронизации статуса "Прочитано" для всех типов чатов (Direct, Group, Admin).
- **Messenger**: Оптимизирован сервис `getUnreadCounts` (добавлены таймауты 20с и условная фильтрация для клиентов/сотрудников для обхода `TimeoutException`).
- **Messenger**: Исправлена ошибка UUID (`admin_chat`) при загрузке информации о чате для клиентов.

## [1.0.6] - 2026-04-14

### Fixed
- **Messenger**: Решена проблема «вечной загрузки» и гонки состояний (Race Condition) при быстром переключении чатов через механизм `LoadId`.
- **Messenger**: Добавлена сетевая устойчивость: все запросы теперь имеют жесткие тайм-ауты (10-15с), что предотвращает зависание UI при плохом соединении.
- **Database**: Устранены ошибки `PostgrestException` (type mismatch `user_role = text`) в триггерах и запросах через явное приведение типов и in-memory фильтрацию.
- **Performance**: Загрузка списка чатов ускорена за счет параллелизации всех компонентов (Direct, Group, Channels, Unread) через `Future.wait`.

### Added
- **UI**: Кнопка «Прокрутка вниз» (Scroll-to-bottom) в чате с автоматическим появлением.
- **Core**: Завершена интеграция Firebase для PUSH-уведомлений на всех платформах.

## [1.0.5] - 2026-04-03

### Fixed
- **Build**: Исправлены ошибки линковки Windows (LNK2019, LNK1104, LNK4099) путем перехода на стандарт C++20 и подключения ATL.

## [1.0.4] - 2026-04-03

### Fixed
- **Network**: Внедрен собственный Reverse Proxy (Cloudflare Workers) для обхода блокировки домена `supabase.co` в РФ. Решена проблема «вечной загрузки» без VPN.

## [1.0.3] - 2026-04-03

### Fixed
- **Messaging**: Исправлено отображение подписей (captions) для файлов и изображений в чате (теперь текст отображается под вложением).
- **UI**: Редизайн диалогового окна отправки файлов (`SendFileDialog`) для лучшей отзывчивости на мобильных устройствах и улучшения эстетики (Telegram-style).
- **UI**: Добавлена поддержка адаптивного сдвига диалога при открытии клавиатуры.

## [1.0.2] - 2026-04-03

### Added
- **Communication**: Внедрена система **Supabase Presence** для отслеживания активных администраторов в чатах (баннер в MessengerScreen).
- **Communication**: Автоматическое отслеживание «Первого ответа» администратора (поле `first_responder_id` в `group_chats`).
- **Communication**: Таблица `profile_notes` и функционал внутренних заметок администратора для профилей клиентов.
- **UI**: Вкладка «Заметки» в `ChatInfoDialog`, видимая только ролям `admin` и `manager`.
- **UI**: Индикация статуса обработки (❓/✅) в списке чатов с аудитом (кто и когда ответил).

### Changed
- **Messaging**: Оптимизирован провайдер `chatPresenceProvider` для работы в реальном времени.
- **Messaging**: Изменен SQL-триггер `handle_message_notification` для корректной маршрутизации уведомлений администраторам.

## [1.0.1] - 2026-04-03

### Changed
- **UI/UX**: Замена фирменного цвета с Purple на Gold (Dark Theme). Изменены файлы `app_theme.dart` и `telegram_colors.dart`.
- **UI/UX**: Обновлены градиенты на экранах логина и регистрации, заменены хардкоды фиолетового цвета.
- **Assets**: Интегрирован новый логотип для Android, iOS, Windows и macOS через `flutter_launcher_icons`.

### Added
- **Core**: Поддержка системных уведомлений на Desktop (Windows, Linux) через пакет `local_notifier`. Уведомления поступают, пока приложение открыто и слушает поток `messages` (Realtime).
