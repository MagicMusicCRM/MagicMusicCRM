# Changelog (v1)

Все локальные изменения в рамках архитектуры v1 фиксируются здесь. Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.5] - 2026-04-03

### Fixed
- **Build**: Исправлены ошибки линковки Windows (LNK2019, LNK1104, LNK4099) путем перехода на стандарт C++20 и подключения ATL.

## [1.0.4] - 2026-04-03

### Fixed
- **Network**: Внедрен собственный Reverse Proxy (Cloudflare Workers) для обхода блокировки домена `supabase.co` в РФ. Решена проблема «вечной загрузки» без VPN.

## [1.0.3] - 2026-04-03

### Fixed
- **Messaging**: Исправлено отображение подписей (captions) для файлов и изображений в чате (теперь текст отображается под вложением).
- **UI**: Редизайн диалогового окна отправки файлов (`SendFileDialog`) для лучшей отзывчивости на мобильных устройствах и улучшения эстетики (Telegram-style).
- **UI**: Добавлена поддержка адаптивного сдвига диалога при открытии клавиатуры.

## [1.0.2] - 2026-04-03

### Added
- **Communication**: Внедрена система **Supabase Presence** для отслеживания активных администраторов в чатах (баннер в MessengerScreen).
- **Communication**: Автоматическое отслеживание «Первого ответа» администратора (поле `first_responder_id` в `group_chats`).
- **Communication**: Таблица `profile_notes` и функционал внутренних заметок администратора для профилей клиентов.
- **UI**: Вкладка «Заметки» в `ChatInfoDialog`, видимая только ролям `admin` и `manager`.
- **UI**: Индикация статуса обработки (❓/✅) в списке чатов с аудитом (кто и когда ответил).

### Changed
- **Messaging**: Оптимизирован провайдер `chatPresenceProvider` для работы в реальном времени.
- **Messaging**: Изменен SQL-триггер `handle_message_notification` для корректной маршрутизации уведомлений администраторам.

## [1.0.1] - 2026-04-03

### Changed
- **UI/UX**: Замена фирменного цвета с Purple на Gold (Dark Theme). Изменены файлы `app_theme.dart` и `telegram_colors.dart`.
- **UI/UX**: Обновлены градиенты на экранах логина и регистрации, заменены хардкоды фиолетового цвета.
- **Assets**: Интегрирован новый логотип для Android, iOS, Windows и macOS через `flutter_launcher_icons`.

### Added
- **Core**: Поддержка системных уведомлений на Desktop (Windows, Linux) через пакет `local_notifier`. Уведомления поступают, пока приложение открыто и слушает поток `messages` (Realtime).
